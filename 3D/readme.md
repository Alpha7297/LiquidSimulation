## 3D 项目说明

这个文件夹保存三维版本的流体仿真程序，初始水块、容器尺寸和虚粒子墙尽量保持一致，方便横向比较不同方法的行为。

另外由于3D计算压力较大，如果有cuda推荐使用cuda，目前默认cpu

### IDP.py

三维 PIC/FLIP+IDP 方法。程序使用 Taichi GGUI 渲染粒子和网格线，右键拖动可以移动相机，`C` 重置相机，`R` 重置仿真。

### wcsph.py

三维弱可压缩 SPH。压力由状态方程计算，虚粒子墙参与密度和压力计算，同时边界位置约束防止粒子穿出容器。流体初始粒子与 `IDP.py` 一致，都是同一块液体区域内每个网格放置 `3^3` 个粒子。

### 交互方式

所有三维程序都支持以下基础操作：

- `Space`：暂停或继续仿真。
- `R`：重置仿真。
- `[`：减小时间步长 `dt`。
- `]`：增大时间步长 `dt`。
- GUI 面板中的 `dt` 滑条：直接调整时间步长。
- 按住鼠标右键拖动：移动相机视角。
- `C`：重置相机视角。

`IDP.py` 额外支持：

- `Left`：减小 `flipRatio`。
- `Right`：增大 `flipRatio`。
- GUI 面板中的 `flipRatio` 滑条：直接调整 PIC/FLIP 混合比例。

`wcsph.py`额外支持：

- `B`：显示或隐藏虚粒子墙。
