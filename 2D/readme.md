## 2D 文件夹说明

这个文件夹中主要保存二维流体仿真的代码文件。

### IDP.py

二维PIC/FLIP+IDP

代码使用交错网格进行速度投影，用粒子携带流体位置和动量，并加入IDP密度修正来缓解局部压缩和体积损失。

### isph.py

二维不可压缩SPH版本。

代码不使用状态方程直接计算压力，而是通过压力泊松方程修正速度场，使速度尽量满足不可压缩条件。

### wcsph.py

二维弱可压缩SPH版本。

代码通过SPH核函数计算粒子密度，再用状态方程得到压力，并使用虚粒子处理边界。

### imgui.ini

Taichi GUI 运行时生成的界面配置文件。

### 交互方式

所有二维程序都支持以下基础操作：

- `Space`：暂停或继续仿真。
- `R`：重置仿真。
- `[`：减小时间步长 `dt`。
- `]`：增大时间步长 `dt`。
- GUI 面板中的 `dt` 滑条：直接调整时间步长。

`IDP.py` 额外支持：

- `Left`：减小 `flipRatio`。
- `Right`：增大 `flipRatio`。
- GUI 面板中的 `flipRatio` 滑条：直接调整 PIC/FLIP 混合比例。
