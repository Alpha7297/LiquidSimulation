## 2D 文件夹说明

这个文件夹中主要保存二维流体仿真的代码文件。

### IDP.py

二维PIC/FLIP+IDP

代码使用交错网格进行速度投影，用粒子携带流体位置和动量，并加入IDP密度修正来缓解局部压缩和体积损失。

### wcsph.py

二维弱可压缩SPH版本。

代码通过SPH核函数计算粒子密度，再用状态方程得到压力，并使用虚粒子处理边界。

### imgui.ini

Taichi GUI 运行时生成的界面配置文件。

### density_compare.md

这个项目比对了几种不同的求解器在同样初始条件下最后的求解结果，主要比对的是密度相对初始时的关系，发现长时模拟下IDP效果明显好于normal

### 交互方式

所有二维程序都支持以下基础操作：

- `Space`：暂停或继续仿真。
- `R`：重置仿真。
- `[`：减小时间步长 `dt`。
- `]`：增大时间步长 `dt`。
- GUI 面板中的 `dt` 滑条：直接调整时间步长。

`IDP.py` 额外支持：

- `Left`：减小 `flipRatio`。
- `Right`：增大 `flipRatio`。
- GUI 面板中的 `flipRatio` 滑条：直接调整 PIC/FLIP 混合比例。
